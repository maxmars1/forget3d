Forget 3D Game Engine

Forget 3D Game Engine is a OpenGL ES framework(still not a engine, ~_~) for Android, Win32,
WinCE platform, it simplifies your OpenGL ES program development, support scene management,
Texture, Camera, Light, Model loader, Font etc.

Release Date:		10-Jun-2009

Current Version:	0.1
Current Authors:	Martin Foo (i25ffz@hotmail.com)

Features in Version 0.1:
- 3D model loader and management
 Current support (Quake II)MD2 model.
- Texture management
 Current support 8 & 24 bit bitmap image file.
